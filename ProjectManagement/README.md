# ArabamFiyat.com - Project Management

Bu klasor, YZTA Bootcamp Scrum sablonundaki kanit yapisini ArabamFiyat.com icin tutar. Proje dokumanlari gercek sprint kararlarini, backlog durumunu ve urun kanitlarini belgelemek icindir; gecmise dair hayali kayit uretilmez.

## Takim ve Urun Bilgileri

| Alan | Bilgi |
| --- | --- |
| Takim adi | Teslimden once ekip tarafindan eklenecek |
| Urun adi | ArabamFiyat.com |
| Problem | Ikinci el arac alici ve saticilarinin benzer ilanlardaki fiyat belirsizligi |
| Cozum | Temizlenmis guncel ilanlardan piyasa analizi, degerleme ve fiyat trendi |
| Hedef kitle | Bireysel arac alici/saticilari ve kucuk olcekli saticilar |
| Backlog | [ProductBacklog.md](ProductBacklog.md) |

Teslimden once bu dosyaya gercek takim uyeleri, GitHub kullanici adlari ve Scrum rolleri eklenmelidir. Roller, ekipteki fiili sorumluluklara gore yazilmalidir.

## Sprint Kanitlari

- [Sprint 1](Sprint1Documents/README.md): veri hatti, cleaning ve ilk piyasa analizi.
- [Sprint 2](Sprint2Documents/README.md): FastAPI, React arayuzu ve tarihsel piyasa takibi.
- [Sprint 3](Sprint3Documents/README.md): ML fiyat tahmini, test, deploy ve final teslimi.

Her sprint klasorundeki `evidence/` alanina gercek board, daily Scrum veya urun ekran goruntuleri eklenmelidir.
