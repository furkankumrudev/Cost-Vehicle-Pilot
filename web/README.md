# ArabamFiyat.com Web Uygulaması

Bu klasör, gerçek FastAPI verisine bağlanan React + Vite arayüzüdür. Eski `frontend/` klasörü statik prototip olarak korunmuştur.

```bat
npm install
npm run dev
```

Geliştirme adresi: `http://127.0.0.1:5173`

API varsayılan olarak `http://127.0.0.1:8000` adresine bağlanır. Farklı bir API adresi için `.env` dosyasına `VITE_API_URL=...` eklenebilir.
